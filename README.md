# Ada Mahkemesi

Bu proje, “Ada Mahkemesi” case çalışması için geliştirilmiştir. Amaç, en ucuz rüşvetle beraat yolunu hesaplamak ve kullanıcıya sade bir arayüz sunmaktır.

## Kurulum

```bash
git clone https://github.com/baturalpozbilir7-lgtm/ada-mahkemesi.git
cd ada-mahkemesi
npm install
npm start
```
